Sprints de codage

Qu'est-ce qu'un sprint de codage?

Un sprint de codage est un rassemblement de gens qui travaillent sur un projet dans le but de travailler sur une tâche particulière relative au projet.

Diriger un sprint de codage

Diriger un sprint de codage lors des Journées de codage est simple :

  1. Donnez une présentation de 10‒15 minutes pour présenter les attentes du sprint.
  2. Séparez les gens en groupes et divisez les tâches à accomplir. Si une tâche est complexe, prévoyez faire un essai à blanc avec une partie du travail pour vous assurer que tout le monde avance dans la bonne direction.
  3. Codez, assistez les participants et répondez aux questions.

Participer à un sprint de codage

Participer à un sprint de codage implique du codage, il vous faudra donc apporter un ordinateur.

Les logiciels suivant devront aussi être installés sur votre ordinateur :

  • Un navigateur Web avec des outils de déboguage, tel que :
    • Firefox with the Firebug (en anglais uniquement) add-on.
    • Google Chrome.
    • Safari.
    • Internet Explorer 10.
  • Un éditeur de texte ou un environnement de développement intégré, tel que :
    • Notepad++.
    • Aptana (en anglais uniquement).
    • Eclipse (en anglais uniquement).
    • Netbeans.
  • Un client git (en anglais uniquement).
  • Node.js (en anglais uniquement).
  • Grunt (en anglais uniquement).
  • Java JRE 7 ou SDK 7 (en anglais uniquement, pour le sprint de codage au sujet de Cordova).

Nous vous recommandons de tout installer avant l'événement. Mais, si jamais vous éprouvez des difficultés à préparer votre ordinateur, nous auront des gens qui pourront vous aider sur place.

Sprints de codage programmés

Nouvelle fonctionalité de pagination pour la Boîte à outils de l'expérience Web

Le but est d'ajouter un plugiciel en jQuery appelé simplePagination.js à la Boîte à outils de l'expérience Web pour gérer la pagination.

  • simplePagination.js (en anglais uniquement)
  • Démo (ZIP, 87 ko, en anglais uniquement)
  • Exemple réel

Nouveau design pour l'interface à onglets

Ce sprint s'adresse à quiconque est intéressé à améliorer l'interface à onglets de la Boîte à outils de l'expérience Web. Nous travaillerons sur le design de nouveaux panneaux d'onglet réactifs, sur le remplacement des codes ringards en JavaScript par des beaux codes en CSS3 et sur la préparation du plugiciel à la version 4.0 de la Boîte à outils.

  • Responsive Web design (en anglais uniquement)
  • CSS-only tabs (en anglais uniquement)

Extension du plugiciel des graphiques

Utilisation et ajout du plugiciel Flot au plugiciel des graphiques de la Boîte à outils de l'expérience Web. Source de données représentée par des tableaux HTML. Utilisation de paramètres CSS, attribut HTML data-* et de paramètres JSON.

  • Présentation pour le sprint de codage au sujet de l'extension du plugiciel des graphiques
  • Flot (en anglais uniquement)
  • Exemples du l'utilisation de Flot (en anglais uniquement)
  • Plugiciels pour Flot(en anglais uniquement)
  • Plugiciel des graphiques (en anglais uniquement)

Variante Apache Cordova (Phonegap) de la Boîte à outils de l'expérience Web pour les applications mobile natives

Le but est de déveloper et maintenir un projet Cordova pour le développement d'applications pour les téléphones et les tablettes et se servant de la Boîte à outils de l'expérience Web comme cadre Web. L'objectif est d'ajouter des fonctionalités multi-platformes pour la manipulation de la caméra, du stockage de données locale, des notifications Push et des autres services qui ne sont pas présentement supportés par le HTML5.

  • Android SDK (en anglais uniquement)
  • Node.js (en anglais uniquement)
  • Apache Cordova (en anglais uniquement)

Rafiner le cache d'application de la Boîte à outils de l'expérience Web

Faisons un sprint afin de s'assurer que la Boîte à outils de l'expérience Web prenne avantage de cette importante fonctionalité du HTML5 pour le stockage hors ligne et pour la performance de chargement.

  • Guide du débutant pour l'utilisation du cache d'application (en anglais uniquement)
  • Faits au sujet du cache d'application (en anglais uniquement)

PJAX pour la Boîte à outils de l'expérience Web

Explorons la possibilité d'ajouter ce plugiciel jQuery à la Boîte à outils de l'expérience Web pour une navigation plus rapide.

  • jquery-pjax (en anglais uniquement)
