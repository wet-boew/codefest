Web Experience Toolkit Codefest 2.0

The Web Experience Toolkit community is proud to be hosting WxT Codefest 2.0.

Join us August 8th and 9th, 2013.

The Web Experience Toolkit (WET) community invites Web coders, designers, accessibility and usability professionals, technical writers to code and design solutions for version 4 of the Web Experience Toolkit.

Codefest will be two days where community will be hosting activities including code sprints, breakout sessions, keynote speakers, various regional satellite events and a UX design jam.

This is a free, open, collaborative event for public servants, WET contributors and interested organizations.

Code sprints

Running a sprint

Please use the following approach to run a sprint:

  • Give a 10‒15-minute introduction explaining what you're hoping to accomplish during the sprint.
  • Split participants into groups and divide up the work. If a task is complex, you should do a dry-run with a small portion of the work to make sure that everyone is on the right track.
  • Code, help participants and answer questions.
